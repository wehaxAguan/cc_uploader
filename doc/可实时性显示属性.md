# UI 数据可实时性显示属性

本文档梳理 Website 和 App 项目中所有业务 UI 数据字段的实时性能力，基于 WebSocket 数据模型判断哪些字段可通过 WS 实时更新。

> **范围**：排除 Hyperliquid 和 MEME 相关模块

## 标记说明

| 标记 | 含义 |
|------|------|
| ✅ WS 实时 | 通过 WebSocket 推送实时更新 |
| 🔄 WS + 计算 | WS 推送源数据 + 本地实时计算 |
| ⏱️ HTTP 轮询 | 定时 HTTP 请求更新 |
| 📥 按需查询 | 用户操作或页面初始化时 HTTP 查询 |
| 🧮 纯本地计算 | 依赖其他实时字段进行本地计算 |
| 📄 静态数据 | 一次性加载，不频繁变化 |
